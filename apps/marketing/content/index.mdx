# Welcome

Welcome to the OpenLoomi docs — a local-first, open-source AI workspace that remembers your work, watches your day, and acts on your behalf when you tap Approve.

## Overview

The docs are organized into four groups:

- **Concepts & guides** (top-level) — what OpenLoomi is, how the desktop app is set up, and how each major surface works. Start with [What is OpenLoomi?](/docs/what-is-openloomi) or [Getting Started](/docs/getting-started).
- **[Use cases](/docs/use-cases)** — real-world workflows that show what OpenLoomi is good at, from proactive decision-making to async standups across time zones.
- **[Reference](/docs/reference)** — power-user material: agent runtimes, embedding providers, credits, keyboard shortcuts, and the public API.
- **[Plugins](/docs/plugins)** — first-party bridges to Claude Code and Codex CLI for users who already work inside another agent shell.

Pick a card below to jump in. New users should head to **Getting Started**; if you're integrating OpenLoomi into an existing agent, jump to **[Plugins](/docs/plugins)** or **[Agent Runtimes](/docs/reference/agent-runtimes)**.
