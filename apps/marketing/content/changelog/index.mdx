# Changelog

Official release notes for OpenLoomi — new feature launches, experience improvements, bug fixes, and breaking changes. Versions are listed in reverse chronological order (latest first) to help you quickly track the product's iteration journey.

## Overview

Each release page covers:

- **New features** — major capabilities shipped in the version
- **Improvements** — UX polish, performance, and small enhancements
- **Bug fixes** — what was broken and is now fixed
- **Breaking changes** — anything that requires migration

OpenLoomi follows [Semantic Versioning](https://semver.org/) loosely: minor versions (0.x.y) may contain breaking changes while the product is pre-1.0. Each release page lists the desktop app, web, and CLI builds that ship together.
